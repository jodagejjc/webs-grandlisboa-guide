# webs-grandlisboa-guide

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面以静态网页形式呈现，涵盖多种主题与功能，旨在提供便捷的信息展示与访问体验。仓库不针对任何特定域名或网站，仅作为内容存档与分发平台。

## 目录说明

```
.
├── index.html          # 主页入口，提供页面导航
├── pages/              # 存放独立 HTML 页面文件的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源文件夹（如有）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本说明文件
```

- `index.html`：仓库主页，列出所有归档页面的链接。
- `pages/`：所有独立 HTML 页面均存放于此。
- `assets/`：可选的 CSS、JavaScript、图片等资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立文件，不依赖外部数据库或后端服务。
- 页面内容可能包含文本、图片、表格、链接等常见网页元素。
- 所有页面均为静态页面，可直接通过浏览器打开或部署到任意静态服务器。

## 维护说明

- 欢迎提交 Issue 报告页面问题或提出改进建议。
- 如需新增页面，请按照现有目录结构放置文件，并在 `index.html` 中添加相应链接。

- 页面内容仅供归档用途，不对其准确性、完整性或适用性作任何明示或暗示的保证。
